# MAHRIO-static
Get started with a static mahrio site

## Instructions

#Regular Installation

* clone this repo
* cd `MAHRIO-static`
* `npm i` to install `mahrio`
* start up the nodejs server with `node index.js`
* navigate to `127.0.0.1:6085`

#Docker Installation

* clone this repo
* cd `MAHRIO-static`
