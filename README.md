# MAHRIO-static
Get started with a static mahrio site

### How to get started

* clone this repo
* cd `MAHRIO-static`

### Running with Docker

* `docker-compose up` to install and startup

* navigate to `127.0.0.1:8081`

### Running without Docker

* `npm i` to install nodejs packages
* `node index.js` to startup node webapp
* navigate to `127.0.0.1:8081`
